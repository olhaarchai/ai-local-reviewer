Focus on the following OWASP Top 10 (2021) categories:
- A01:2021 Broken Access Control — missing auth checks, IDOR, privilege escalation
- A02:2021 Cryptographic Failures — plaintext secrets, weak hashing, hardcoded keys
- A03:2021 Injection — SQL, command, NoSQL injection via string concatenation
- A05:2021 Security Misconfiguration — debug flags, open CORS, exposed stack traces
- A07:2021 Identification and Authentication Failures — missing token validation, insecure sessions
- A10:2021 Server-Side Request Forgery — unvalidated URLs passed to HTTP clients
